# Athleisure as the New Luxury: A Data-Driven Look at Fashion Trends

This project analyzes the evolution of athleisure from everyday comfort wear to a new form of luxury, using real pricing and popularity data from top brands including Lululemon, Alo Yoga, SKIMS, Gymshark, and Fabletics.

##  Project Overview
- **Objective:** Explore how consumer interest and brand pricing indicate athleisure’s transformation into a luxury segment.
- **Tools Used:** Python, Pandas, Seaborn, Matplotlib, Google Trends, Particl dataset.
- **Key Analyses:**
  - Popularity growth via Google Trends
  - Price distribution comparisons across brands
  - Best-seller product analysis to test if customers favor premium pricing

## Repository Structure
/data — cleaned and processed CSVs and the generated CSVs with the average and standardized price distributions

/notebooks — step-by-step data cleaning with visualizations

/visualizations — charts used in analysis as PNGs for easy blog integration

##  Insights
- Premium athleisure brands show luxury-tier pricing trends.
- Best-sellers often align with higher-priced categories, suggesting value perception beyond affordability.
- Comfort is no longer casual — it’s a new form of status.
